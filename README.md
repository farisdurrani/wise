# wise
Ask and we give you wise quotes from your own books.

# Frontend
## Prerequisites
- Node.js v18

## How to start
```
cd frontend
npm install
npm start
```
Open the app at http://localhost:3000

# Backend
## Prerequisites
- Python 3.10

## How to start
```
pip install -r requirements.txt
python main.py
```
Open the app at http://localhost:8000
